# 📜 Parancslista

Itt megtalálod a Ghosty Bot összes parancsát kategóriákra bontva. Minden parancs slash (`/`) alapú.

---

## ℹ️ Információs Parancsok

| Parancs | Leírás |
| :--- | :--- |
| `/botinfo` | Megjeleníti a bot legfontosabb információit. |
| `/serverinfo` | Részletes statisztikák a jelenlegi szerverről. |
| `/userinfo` | Megjeleníti egy adott felhasználó adatait. |
| `/inviteinfo` | Megmutatja egy Discord meghívó részleteit. |
| `/channelinfo` | Információk az adott csatornáról. |
| `/roleinfo` | Egy adott szerepkör (role) részletei. |
| `/rolelist` | Kilistázza a szerveren található összes rangot. |
| `/channellist` | Kilistázza a szerver összes csatornáját. |
| `/ping` | Ellenőrzi a bot válaszidejét (latencia). |

---

## 🛡️ Moderációs Parancsok

- `/kick` – Felhasználó kirúgása.
- `/ban` – Felhasználó végleges kitiltása.
- `/unban` – Kitiltás feloldása.
- `/addrole` – Szerepkör hozzáadása.
- `/removerole` – Szerepkör elvétele.
- `/clear` – Üzenetek törlése.
- `/purge` – Üzenetek törlése.
- `/mute` – Felhasználó némítása.
- `/unmute` – Némítás feloldása.
