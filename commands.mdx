Itt megtalálod a Ghosty Bot összes parancsát kategóriákra bontva. Vannak / és ,\> parancsok is.

## Szerver Kezelős Parancsok

| Slash | Prefix | Leírás |
| --- | --- | --- |
| /channel-create | ,\>channel create | Csatorna létrehozása. |
| /channel-delete | ,\>channel delete | Csatorna törlése. |
| /role-create | ,\>role create | Rang létrehozása. |
| /role-delete | ,\>role delete | Rang törlése. |
