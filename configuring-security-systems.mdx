## Antinuke

**Az Engdélyezéshez:** /setup antinuke True\
**A Letiltáshoz:** /setup antinuke False

**A Rendszer Konfigolása:** Az /antinuke config parancsot kell használni.\
\
**Paraméterek:**\
\
max\_actions Maximális műveletek száma, hogy a bot bűntetést adjon ki a felhasználóra aki nuke-ol.\
\
time\_window Időablak másodpercben, hogy mennyi másodperc alatt hajtodik végre az antinuke rendszer.\
\
punishment Bűntetés típusa.\
**Típusok:** Kitiltás, Kirúgás, Rangok Eltávolítása\
\
**A Whitelisthez Adás:** /antinuke whitelist felhasználó akit hozzá szeretnél adni a whitelisthez.\
**A Whitelistből Törlés:** /antinuke unwhitelist felhasználó akit törölni szeretnél a whitelistből.\
\
**Állapot Megtekintése:** /antinuke status

## Antiraid

**Az Engdélyezéshez:** /setup antiraid True\
**A Letiltáshoz:** /setup antiraid False

**A Rendszer Konfigolása:** Az /antiraid config parancsot kell használni.\
\
**Paraméterek:**\
\
max\_joins Maximális belépések száma, hogy a bot bűntetést adjon ki a felhasználóra aki raid-el.\
\
join\_window Időablak másodpercben, hogy mennyi másodperc alatt hajtodik végre az antiraid rendszer.\
\
max\_messages Maximális üzenetek száma, mennyit írhat egy felhasználó másodpercen ként.\
\
message\_window Üzenet időablak másodpercben, mennyi másodperc alatt érkezik annyi üzenet hogy az antiraid rendszer aktíbája magát.\
\
**Szerver Lezárása Raid Esetén:** /antiraid lockdown\
\
**Állapot Megtekintése:** /antiraid status

### AntiAlt

**Az Engdélyezéshez:** /antialt enable\
**A Letiltáshoz:** /antialt disable\
\
**A Rendszer Konfigolása:** Az /antialt settings paranccsal kell használni.\
\
**Paraméterek:**\
\
setup Amit akarsz módosítandó beállítás.\
**Típusok:** Bekapcsolás, Kikapcsolás, Minimum Fiókkor, Művelet, Profilkép kötelző, Log Csatorna, Büntett fiókok törlése\
\
days Minimum fiókkor napokban. (Csak akkor kell ezt használni ha Minimum fiókkkor van be nyomva.) vagy az /antialt setage paranccsal is beállítható.\
\
action Amit végrehajtson a rendszer, amikor aktíválódik. (Csak akkor kell ezt használni ha Művelet van be nyomva.) vagy az /antialt setaction paranccsal is beállítható.\
**Típsuok:** Kirúgás, Kitiltás, Némitás\
\
value A profilkép ellenörzés, állítása. (Csak akkor kell ezt használni ha Profilkép kötelező van be nyomva.)\
\
channel A log csatorna helye (Csak akkor kell ezt használni ha Log csatorna van be nyomva.) vagy az /antialt setlog paranccsal is beállítható.\
\
**A Whitelisthez Adás:** /antialt whitelist felhasználó akit hozzá szeretnél adni a whitelisthez.\
**A Whitelistből Törlés:** /antialt removewhitelist felhasználó akit törölni szeretnél a whitelistből.\
\
**Állapot Megtekintése:** /antialt status
