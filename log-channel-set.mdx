---
title: "Log Csatorna Beállítása"
---

**Beállítás:** /setup logchannel type channel\
\
**Paraméterek:**\
\
type - A típus amit bekarasz állítani.\
\
channel - A csatorna ahova megy az a típusú log.

### Típusok:

- Emoji
- Matrica
- Üzenet
- Moderáció
- Szerver
- Integráció
- Csatorna
- Gondolatmenet
- Hang (Nem érdemes beállítani)
- Rang
- Automod
- Felhasználó
- Webhook
- Ticket
