A Ghosty bot részletes naplózási rendszert kínál, amellyel minden eseményt nyomon követhetsz a szervereden. Különböző típusú eseményeket akár különböző csatornákba is irányíthatsz.

---

## A Beállító Parancs

Használd az alábbi slash parancsot a logolás konfigurálásához:

<CodeGroup>

```bash Parancs
/setup logchannel type:[típus] channel:[#csatorna]
```

</CodeGroup>
